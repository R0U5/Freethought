LLM Fine-Tuning Engine

A high-efficiency fine-tuning framework for transformer models (e.g., Phi-2) with support for LoRA, adaptive training dynamics, and structured data handling. Includes features like memory-aware tokenization, dynamic early stopping, error-aware recovery logic, and seamless multi-mode support (SFT, DPO). Designed for local GPU optimization and large-scale dataset training with minimal manual intervention.

Intelligent Response Evaluator

Runs structured QA batches across categories like logic, math, science, and language. Includes primed examples and answer cleaning routines to simulate real-world chat deployment. Helps monitor LLM performance drift and detect hallucinations or weak spots post-training. Also uses deterministic tests on a range of academic and commonsense questions. Includes auto-grading logic and a final radar chart to visualize model performance across categories.

Visual Training Diagnostics

Standalone GUI tool to analyze training logs. Features interactive loss curve plotting, EMA smoothing, variance scoring, and trend classification. Automatically grades model progress and predicts ideal stopping points.
